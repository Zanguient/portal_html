# README #

Módulo que:
 - gerencia o layout da tela de impressão de relatórios de forma genérica;

 
 
### Pra Que Serve Este Módulo? ###

Gerenciar completamente a tela de impressão de um relatório para diversas telas do Portal.  



### O que é necessário para usar o módulo? ###

É um index "à parte" do projeto completo. Necessita que seja enviado via URL os dados para consulta direta na API
