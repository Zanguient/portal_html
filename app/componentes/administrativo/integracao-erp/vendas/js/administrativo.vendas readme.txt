# README #

Módulo que:
 - gerencia o layout da tela de vendas de Integração ERP (app/componentes/administrativo/integracao-erp/vendas/index.html);
 - permite a consulta das vendas já importados da base do cliente
 - permite a importação das vendas da base do cliente para a base da atoscapital
 

### Pra Que Serve Este Módulo? ###

Gerenciar completamente a tela de tela de vendas de Integração ERP (Administrativo). 



### O que é necessário para usar o módulo? ###

Como esse é um controller filho do controller principal "appCtrl" (app.js), ele herda todos os módulos importados por ele.
Além deles, é necessária a importação dos seguintes módulos externos.

 |----------------------------------------|---------------------------------------------------|-------------------------------------------|
 |                MODULO                  |                   DESCRIÇÃO                       |         IMPORT (no app/index.html)        |
 |----------------------------------------|---------------------------------------------------|-------------------------------------------|
 |                       
 |----------------------------------------|---------------------------------------------------|-------------------------------------------|


 
### Outras depêndencias ###

  * Evento "mudancaDeRota", proveniente do "appCtrl", para de fato modificar a rota/estado
  * $scope.token do "appCtrl"
  * Emite evento "acessouTela" no init do controller para que seja feito o log de acesso de tela
  * Evento "acessoDeTelaNotificado", proveniente do "appCtrl", para de fato exibir a tela e fazer requisições 
	=> importante para o log correto de requisições HTTP, ao qual identifica corretamente a (tela de) origem
  * $scope.grupoempresa, do "appCtrl" : grupo empresa informado pelo usuário na barra administrativa
  * Evento "alterouGrupoEmpresa", emitido pelo controller "appCtrl", para identificar a mudança do $scope.grupoempresa
  * APIS:
    - card/vendaserp
	- card/tbrecebimentovenda
	
 
 
 ### Interação com a WEB API ###
 
  * card/vendaserp
	- POST : importa vendas da base da empresa para a base da atoscapital
			 Espera o JSON { data : string }, que é a data do título
			 
        - PATCH : importa vendas a partir do arquivo CSV (upload)	
	
   
  * card/tbrecebimentovenda
        - GET : carrega as vendas já importadas (coleção 2)
			[
			  { empresa : { nu_cnpj : string, ds_fantasia : string, filial : string ou null },
				nrNSU : string,
				dtVenda : datetime ou null,
				tbAdquirente : { cdAdquirente : int, nmAdquirente : string },
				dsBandeira : string,
				vlVenda : double ou null,
				qtParcelas : int ou null,
				cdERP : string ou null,
				dtAjuste : datetime ou null,
				cdSacado : string ou null,
				conciliado : bool,
 				dsMensagem: string ou null,// indica se precisa de intervenção manual no ERP
			  }
			]	
		   Recebe em Totais: - "valorTotal" : double // valor total parcela
				     - "totalCorrigidos" : int // total de vendas corrigidas
		      		     - "totalConciliados" : int // total de vendas conciliadas
				     - "totalCorrecaoManual" : int // total de vendas que precisam de correção manual
