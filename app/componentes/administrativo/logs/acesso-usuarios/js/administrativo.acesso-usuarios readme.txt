# README #

Módulo que:
 - gerencia o layout da tela de acesso de usuários de Logs (app/componentes/administrativo/logs/acesso-usuarios/index.html);
 

### Pra Que Serve Este Módulo? ###

Gerenciar completamente a tela de acesso de usuários de Logs.  



### O que é necessário para usar o módulo? ###

Como esse é um controller filho do controller principal "appCtrl" (app.js), ele herda todos os módulos importados por ele.
Não é necessário nenhum outro módulo externo.

 
### Outras depêndencias ###

  * Evento "mudancaDeRota", proveniente do "appCtrl", para de fato modificar a rota/estado
  * $scope.token do "appCtrl"
  * Emite evento "acessouTela" no init do controller para que seja feito o log de acesso de tela
  * APIS:
    -  administracao/logacesso
	
 
 
 ### Interação com a WEB API ###
 
  * administracao/logacesso
	- GET : obtém a lista de usuários baseado no log de acesso aos controllers (coleção 2)
			{
				webpagesusers = new { id_users = int,
									  ds_login = string
									},
				controller = new { id_controller = int,
								   ds_controller = string,
								 },
				dtAcesso = datetime,
				flMobile = boolean,
				dsUserAgent = string
			 }
