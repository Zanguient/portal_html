# README #

Layout principal, acessado após validação do login do usuário. 


### Referência ###

O layout foi feito a partir do template Metronic, v3.7, a partir do arquivo "metronic_v3.7/theme/templates/admin3/index.html".



### Módulo AngularJS ###

	O index.html faz uso do módulo "AtosCapital", definido no arquivo, por essa razão as libs correspondentes devem ser importadas:
     * angular.min.js
	 * app.js
	

### Outros layouts que são exibidos nessa index.html ###
	
   Todos os arquivos index.html que podem ser localizados interno a pasta app/componentes. 
   Portanto, todos os scripts e CSSs utilizados para estes layouts devem ser referenciados na index.html documentada neste arquivo.
